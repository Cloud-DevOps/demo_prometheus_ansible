Prometheus Live Demo

This is a live demo of the Prometheus monitoring system.

Compontents included:

  • Prometheus - Main server, you're looking at it
  • Node Exporter - Machine metrics
  • Alertmanager - Alert aggregation and routing
  • Push Gateway - Batch jobs push metrics here

This is a simple single-machine setup. The Ansible cookbooks are available.
